# AVL树

定义：

-   AVL树：高度平衡的二叉搜索树（任意节点的两个子树的高度差$\le$ 1）

-   平衡因子：某个节点的左子树高度与右子树高度的差

    因此，AVL树又可作如下定义：任意节点平衡因子的绝对值小于等于1的二叉搜索树



当插入或删除节点导致不平衡时，AVL通过旋转节点来重新平衡节点。
