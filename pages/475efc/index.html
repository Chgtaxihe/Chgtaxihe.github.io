# a/b%c=a%(b∗c)/ba/b \% c = a \% (b*c)/ba/b%c=a%(b∗c)/b

前提: a/ba/ba/b为整数

证明:

存在整数kkk使得

ab%c=ab−kc\frac{a}{b}\%c=\frac{a}{b}-kcba​%c=ba​−kc

因此

ab%c=a−kbcb\frac{a}{b}\%c=\frac{a-kbc}{b}ba​%c=ba−kbc​

注意上面的式子用的是等号而非同余

根据上述定理，可以使用以下代码求解本题

ll get_sum(ll p, ll t){ // 求 1, p, p^2 ... p^t 的和

    ans = ans / (p-1);
    return ans % mod;
